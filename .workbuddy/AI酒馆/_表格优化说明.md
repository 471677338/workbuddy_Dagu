# 《魔王神官和勇者美少女》表格优化说明

## 📊 优化概述
基于原`table_data (1).json`文件，针对小说/RP场景**全面优化**，更适合SillyTavern深度角色扮演。

## 🔍 主要改进点

### 1. **表格内容深度增强**
- **原版**：简单描述
- **优化版**：增加了 **情感发展、剧情影响、优先级、信任程度**等维度
- **效果**：AI能更好地理解角色互动和剧情推进

### 2. **角色表优化**
```
原：
["角色名","身体特征","性格","职业","爱好","喜欢的事物","住所","其他重要信息"]

优化：
["角色名","身体特征","性格","职业/身份","关系网络","魔力/能力特质"]
```
- **增加关系网络**：让AI清楚角色间如何关联
- **整合"魔力/能力特质"**：奇幻世界观的核心
- **删减次要信息**：浓缩关键信息

### 3. **新增表格内容**
| 新增表格 | 作用 |
|---------|------|
| **世界观设定补充** | 统一背景设定，防止AI编造矛盾 |
| **角色互动关系** | 聚焦角色间的情感动态和互动模式 |
| **当前任务与目标** | 明确剧情方向和优先级 |

### 4. **情感线与剧情线分开**
```
原版：将情感与事件混合在"重要事件历史表格"
优化版：分拆为：
 - 重要事件历史与情感发展（侧重感受）
 - 世界观设定补充（侧重客观）
```

### 5. **数据增强**
- **角色**：从4个→6个（增加**未知黑袍人**）
- **物品**：从5个→10个（增加了**能力、契约、魔法系统**）
- **事件**：从9行→8行，但**每行信息更丰富**
